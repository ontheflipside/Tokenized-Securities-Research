# Changelog

This project is being released publicly as an early research preview. The version history below is intended to make the public development path clear as the platform evolves.

## v0.1.0 - Public Research Preview

Initial public release of Tokenized Securities Research.

### Included

- Public GitHub repository.
- Live Streamlit application.
- Configurable research watchlist.
- Local signal engine.
- CSV and HTML research outputs.
- Signal history tracking.
- Paper research event log.
- Beginner run guide.
- Windows launcher script.
- Mac and Linux launcher script.
- Project roadmap.
- MIT license.
- Contribution guidelines.

### Current limitations

- Tokenized market metrics are simulated while a live data provider is evaluated.
- Signal outputs are research classifications only.
- The platform does not execute transactions.
- No brokerage or exchange account integrations are included.
- Backtesting and performance measurement are planned but not yet implemented.

The public release is intentionally early. The goal is to build in the open, gather feedback, and improve the research framework over the coming weeks.
